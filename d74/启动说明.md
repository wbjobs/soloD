# 快速启动指南

## 前置条件

1. 确保已安装 Node.js: https://nodejs.org/
2. 确保已安装 MongoDB: https://www.mongodb.com/try/download/community

## 启动步骤

### 步骤 1: 启动 MongoDB

**方法一：Windows 服务（推荐）**
```powershell
# 以管理员身份运行 PowerShell
net start MongoDB
```

**方法二：直接运行**
```powershell
# 找到 MongoDB 的 bin 目录（根据你的安装路径调整）
cd "C:\Program Files\MongoDB\Server\7.0\bin"
.\mongod.exe
```

### 步骤 2: 启动后端服务器（新终端）

```powershell
cd d74\server
npm install
npm start
```

看到 "MongoDB 连接成功" 和 "服务器运行在 http://localhost:3001" 表示成功。

### 步骤 3: 启动前端服务器（新终端）

```powershell
cd d74
npx serve . -p 3000
```

### 步骤 4: 访问应用

打开浏览器访问: http://localhost:3000

## 功能测试

1. 点击「生成地形」按钮 - 应该看到地形生成
2. 点击「💾 保存场景」按钮 - 应该提示保存成功
3. 点击「清空所有」按钮 - 场景应该被清空
4. 点击「📂 加载场景」按钮 - 选择刚才保存的场景加载

## 常见问题

### 问题：MongoDB 连接失败
**解决**：确保 MongoDB 正在运行，检查端口 27017 是否被占用

### 问题：前端连接后端失败
**解决**：确保后端服务器已启动，检查 main.js 中的 API 地址是否正确

### 问题：端口被占用
**解决**：修改 server.js 中的端口号，或者关闭占用端口的程序

## 数据存储位置

MongoDB 默认数据目录：`C:\data\db`
