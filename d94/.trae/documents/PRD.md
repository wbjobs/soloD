# 社交媒体关系图谱工具 - 产品需求文档

## 1. 产品概述

社交媒体关系图谱工具是一个用于可视化展示社交网络用户关系的Web应用。通过抓取Twitter/GitHub等平台的用户关注列表，构建有向图数据并以力导向图形式展示，帮助用户直观理解社交网络结构。

- 主要用途：可视化分析社交媒体用户关系网络、发现社群结构、追踪影响力传播路径
- 目标用户：数据分析人员、社交媒体研究者、普通用户
- 产品价值：将抽象的社交关系转化为直观的图形展示，降低网络分析门槛

## 2. 核心功能

### 2.1 功能模块

1. **图谱展示页**：用户输入、力导向图渲染、节点交互
2. **控制面板**：平台选择（Twitter/GitHub）、用户名输入、抓取控制
3. **信息面板**：节点详情展示、统计信息（节点数、边数）

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 图谱展示页 | 顶部导航栏 | 应用标题、平台切换按钮 |
| 图谱展示页 | 输入控制区 | 用户名输入框、抓取按钮、加载状态 |
| 图谱展示页 | 力导向图区域 | D3.js渲染的交互式关系图谱、节点拖拽、缩放 |
| 图谱展示页 | 侧边信息栏 | 选中节点详情、网络统计数据 |

## 3. 核心流程

用户输入用户名 → 选择平台类型 → 点击抓取按钮 → 后端获取模拟数据 → 前端渲染力导向图 → 用户交互探索图谱

```mermaid

    A["用户输入用户名"] --> B["选择平台(Twitter/GitHub)"]

    C --> D["后端API返回模拟数据"]
    D --> E["前端渲染力导向图"]
    E --> F["用户拖拽/点击节点交互"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝渐变 (#1a1a2e → #16213e)，体现科技感和专业性
- **辅助色**：青色节点 (#00d4ff)、橙色高亮 (#ff6b35)、紫色连线 (#9d4edd)
- **按钮风格**：圆角矩形、悬浮发光效果、渐变背景
- **字体**：标题使用 'Orbitron' 科技感字体，正文使用 'Inter' 清晰易读
- **布局风格**：深色主题、玻璃拟态卡片、全屏画布布局
- **图标风格**：线性图标、霓虹发光效果

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 图谱展示页 | 导航栏 | 渐变背景、发光logo、平台切换下拉框 |
| 图谱展示页 | 控制区 | 玻璃拟态卡片、发光输入框、脉冲动画按钮 |
| 图谱展示页 | 图谱区域 | 深色背景、发光节点、流动连线、网格纹理 |
| 图谱展示页 | 信息栏 | 半透明侧边栏、滚动动画、数据卡片 |

### 4.3 响应式设计

- 桌面端优先设计（1200px+）
- 平板端：信息栏折叠为底部面板
- 移动端：控制区堆叠布局，图谱全屏展示
- 触摸设备优化节点点击区域

### 4.4 动画与交互

- 页面加载：元素渐入动画（staggered reveal）
- 节点悬浮：发光增强、尺寸放大
- 连线动画：脉冲流动效果
- 图谱初始化：弹性展开动画
- 数据更新：平滑过渡形变
