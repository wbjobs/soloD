## 1. 产品概述

安全报告辅助工具，帮助安全工程师自动化分析Nmap扫描结果并生成专业的漏洞报告。
- 核心功能：解析Nmap XML扫描结果，通过Ollama Llama 3模型生成Markdown格式的漏洞描述和修复建议
- 目标用户：安全工程师、渗透测试人员、网络安全分析师

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 安全工程师 | 无需注册 | 上传XML文件、查看分析结果、导出报告 |

### 2.2 功能模块
1. **主页面**: 文件上传区域、实时分析结果展示
2. **结果展示**: Markdown渲染、流式输出动画、复制/导出功能

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 文件上传 | 支持拖拽或点击上传Nmap XML文件，显示文件大小和上传状态 |
| 主页面 | 分析状态 | 显示解析进度、LLM调用状态、流式输出过程 |
| 主页面 | 结果展示 | Markdown格式渲染漏洞描述和修复建议，支持复制和导出 |

## 3. 核心流程

用户上传Nmap XML扫描文件 → 后端解析XML提取端口和服务信息 → 构建提示词发送给Ollama Llama 3模型 → 流式接收模型响应 → 前端实时渲染Markdown结果 → 用户可复制或导出报告

```mermaid
flowchart LR
    A["用户上传XML文件"] --> B["后端解析XML"]
    B --> C["提取端口/服务/漏洞信息"]
    C --> D["构建LLM提示词"]
    D --> E["调用Ollama Llama 3"]
    E --> F["流式返回Markdown"]
    F --> G["前端实时渲染"]
    G --> H["用户复制/导出报告"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰蓝 (#1e293b) 搭配亮青色 (#06b6d4) 作为强调色
- 按钮风格：圆角矩形，悬停时有微妙发光效果
- 字体：JetBrains Mono（代码感）搭配 Noto Sans SC（可读性）
- 布局风格：深色主题，卡片式布局，左侧上传区右侧结果区
- 图标风格：简约线性图标，使用 Lucide React

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 主页面 | 文件上传区域 | 拖拽边框、文件图标、上传按钮、进度条 |
| 主页面 | 分析状态 | 加载动画、状态文字、流式输出指示器 |
| 主页面 | 结果展示 | Markdown渲染器、代码高亮、复制按钮、导出按钮 |

### 4.3 响应式
- 桌面端：左右分栏布局（上传区40%，结果区60%）
- 移动端：上下堆叠布局，优化触摸区域
- 流式输出在移动端保持良好的可读性

### 4.4 动效设计
- 文件拖拽时边框发光动画
- 上传成功的确认动画
- 流式输出的打字机效果
- Markdown元素的渐入显示
